Variables & Data Types

ในโลกของการเขียนโปรแกรม "ตัวแปร" (Variable) คือกล่องที่เราใช้เก็บข้อมูล โดยมี "ชนิดข้อมูล" (Data Type) เป็นตัวระบุว่ากล่องใบนั้นมีขนาดเท่าไหร่และใส่ของประเภทไหนได้บ้าง

📊 เปรียบเทียบขนาดพื้นที่จัดเก็บในหน่วยความจำ (Bytes)

ชนิดข้อมูลแต่ละประเภทใช้พื้นที่ในการจัดเก็บต่างกัน การเลือกใช้ให้เหมาะสมจะช่วยให้โปรแกรมทำงานได้มีประสิทธิภาพ กราฟด้านล่างแสดงให้เห็นถึงความแตกต่างของขนาดกล่องข้อมูลแต่ละชนิด

🗂️ เจาะลึกชนิดข้อมูลพื้นฐาน

นอกจากขนาดพื้นที่แล้ว ชนิดข้อมูลยังเป็นตัวกำหนดรูปแบบของข้อมูลที่อนุญาตให้เก็บในตัวแปรนั้นๆ มาดูกันว่าแต่ละประเภทมีลักษณะอย่างไร

🔢

int

(Integer)
  • ประเภท: เลขจำนวนเต็ม
  • ตัวอย่าง: 10, -5, 2024
  • Syntax: int age = 25;
📏

float

(Floating Point)
  • ประเภท: เลขทศนิยม
  • ตัวอย่าง: 3.14, 99.9
  • Syntax: float price = 199.50;
🔠

char

(Character)
  • ประเภท: ตัวอักษร 1 ตัว
  • ตัวอย่าง: 'A', 'z', '#'
  • Syntax: char grade = 'A';
🔬

double

(High Precision)
  • ประเภท: ทศนิยมละเอียดสูง
  • ตัวอย่าง: 0.000000123
  • Syntax: double pi = 3.14159;

✅ สิ่งที่ทำได้ (Correct)

  • 1.
    ขึ้นต้นด้วยตัวอักษร

    score, Name, player1

  • 2.
    ใช้ Underscore ได้

    total_price, _status

  • 3.
    Case Sensitive

    ตัวพิมพ์เล็ก/ใหญ่มีผล age ถือว่าเป็นคนละกล่องกับ Age

  • 4.
    ความหมายชัดเจน

    เช่น student_id สื่อความหมายดีกว่าใช้แค่ s

❌ สิ่งที่ห้ามทำ (Incorrect)

  • 1.
    ห้ามขึ้นต้นด้วยตัวเลข

    1player

  • 2.
    ห้ามมีเว้นวรรค

    item price

  • 3.
    ห้ามใช้สัญลักษณ์พิเศษ

    ห้ามใช้ @, #, $, % ยกเว้น _ เท่านั้น

  • 4.
    ห้ามใช้คำสงวน (Reserved Words)

    เช่น int, float, return เพราะเป็นคำสั่งระบบ

💡

สรุปใจความสำคัญ (Key Takeaways)

1. เลือกให้ถูกขนาด

เลือกกล่องให้เหมาะกับสิ่งที่จะเก็บ เช่น อายุใช้ int เกรดเฉลี่ยใช้ float

2. สื่อความหมาย

ตั้งชื่อตัวแปรเปรียบเหมือนป้ายกำกับกล่อง ยิ่งชัดเจน ยิ่งช่วยให้ตัวเราและเพื่อนร่วมทีมทำงานได้ง่ายขึ้น

3. ประกาศก่อนใช้

ในภาษา C (Static Typing) เราต้องบอกระบบก่อนเสมอว่ากล่องใบนี้ชื่ออะไร และมีชนิดข้อมูลแบบไหน
